# Установка

```bash
mkdir pebbles
composer create-project pebblesengine/skeleton pebbles
```

# Использование

```bash
cd pebbles
php -S 127.0.0.1:8000 index.php
```

Приложение будет доступно по адресу `localhost:8000` в браузере.
